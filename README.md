### cjoop-ad - 行政区划数据抓取工具
# Installation

## Java Requirements
* 在你的电脑上需要有JDK1.7+的环境.

## Maven Requirements
* 在你的电脑上需要安装maven环境.

#### Install with mvn

```sh
$ mvn package
```
target会生成相应的cjoop-ad-crawler-0.0.1-SNAPSHOT-bin.zip压缩包,复制到任意位置,解压后运行startup.bat文件启动程序.
